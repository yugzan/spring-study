Account Manager for Jongo(MongoDB)
===================


How to Start
-------------------
####Step 1.
> cd  /main/scripts/
> npm install
> node insert_mongodb.js

####Step 2.
> **[$~/ProjectPath/]** gradle clean build
> **[$~/ProjectPath/]** java -jar build/libs/project.jar

----------


Note
-------------
A template for Spring Boot Using Jongo(MongoDB).
